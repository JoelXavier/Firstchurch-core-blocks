# Design System Style Guide

Our design language prioritizes **elegance, professionalism, and invitation**. Every token is curated to support these pillars for the Church identity.

---

## 3. Spacing & Layout
We use a **Rule of 8s** scale efficiently mapped to T-shirt sizes.

### Spacing Scale

| Alias (T-Shirt) | Token | Value |
|-----------------|-------|-------|
| `--spacing-xs` | `--spacing--08` | 0.5rem (8px) |
| `--spacing-sm` | `--spacing--16` | 1rem (16px) |
| `--spacing-md` | `--spacing--24` | 1.5rem (24px) |
| `--spacing-lg` | `--spacing--40` | 2.5rem (40px) |
| `--spacing-xl` | `--spacing--64` | 4rem (64px) |
| `--spacing-2xl`| `--spacing--80` | 5rem (80px) |

### Grid System
A fluid grid that adapts to 4 main breakpoints.

| Aspect | Mobile | Tablet | Desktop |
|--------|--------|--------|---------|
| **Columns** | 4 | 8 | 12 |
| **Gutter** | 16px | 24px | 32px |
| **Margin/Side**| 18px | 24px | 70px |
