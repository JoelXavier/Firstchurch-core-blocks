# Iconography

Icons in our system are designed to be simple, elegant, and communicative. They should be used sparingly to enhance navigation and comprehension without cluttering the interface.

## Style Guidelines

*   **Style**: Outline / Strokeline (Preferred) or Solid (Active States).
*   **Stroke Width**: 1.5px or 2px (Consistent).
*   **Color**: Inherit from text (`currentColor`) or specific Semantic Colors.
*   **Corner Radius**: Slightly rounded (matches spacing tokens).

## Sizing Standards

| Size Token | Pixel Value | Usage |
|------------|-------------|-------|
| `Small` | 16px | Inline text, meta data |
| `Medium` | 24px | Navigation, Buttons (Default) |
| `Large` | 32px | Section Headers, Feature Cards |
| `Hero` | 48px | Featured Areas |
